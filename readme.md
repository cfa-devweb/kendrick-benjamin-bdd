# Library database
***
## General information
This project was realized within the framework of an exercise on the relational database by using the merise method. 
The goal was the management of the books and the reservation of those.
The library must be able to restore data, import, save lists of books in CSV format.

## Technologies and langages
The technologies used in this project are SQL and bash script. 
We coded with MySQL WorkBench and MySQL Server for the SQL, and with a text editor and GIT bash for the bash script.

A list of technologies used within the project:
* [MySQL WorkBench]: Version 8.0.23
* [MySQL serveur]
* [SQL lagngage]: Version 15.0.2000.5
* [shell script]
* [GIT bash]:  Version 2.31.0
