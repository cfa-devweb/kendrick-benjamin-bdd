#### SQL SCRIPT ET SHELL SCRIPT ####

Ce projet à pour but de créer une base de donnée permettant la gestion de livre et la réservation de ceux-ci.

Les technologies utilisé dans ce projet sont SQL et du bash script. Nous avons codé avec MySQL WorkBench et MySQL Server pour le SQL, et avec un éditeur de texte et GIT bash pour les script bash.
